## 🌟 Overview

**SafeHer** is a modern full-stack web application focused on women's digital safety. The platform helps users identify cyber threats, learn safe online practices, assess their cyber security posture, report incidents anonymously, and receive AI-powered guidance against cyber crimes.

The application combines cybersecurity education with artificial intelligence to provide practical solutions for threats such as cyberstalking, online harassment, image abuse, sextortion, phishing attacks, and identity theft.
